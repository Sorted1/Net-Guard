
# NetGuard

A Easy Way To Manage Your Network
## Features

- On The Fly Server Networking
- ipconfig editing
- rate limiting 
- ip whitelisting


## Setup

```bash
  sudo pip3 install -r requirements.txt
  sudo python3 main.py
```
